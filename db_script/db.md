# Database Scripts

This folder contains all the database files for the Organizational Knowledge Gap Intelligence Platform.

## Files

- database_backup.sql
  - Complete PostgreSQL database backup.

- knowledge gap.sql
  - Milestone 1 database schema and sample data.

- milestone2_db
  - Milestone 2 database scripts.

- milestone3_db
  - Milestone 3 database scripts.

## Database

PostgreSQL

## Steps to Run

1. Create a PostgreSQL database.
2. Open pgAdmin 4.
3. Execute the SQL scripts.
4. Verify that all tables are created successfully.
